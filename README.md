# Savage Media Productions — Vercel Ready

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Create a new GitHub repo and upload this whole folder.
2. Go to Vercel and import the repo.
3. Framework preset: **Vite**
4. Build command: `npm run build`
5. Output directory: `dist`
6. Deploy.
